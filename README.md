# NYC Insight  
### Real-Time New York News & Video Aggregation Platform

> A real-time content aggregation dashboard delivering the latest New York City news articles and YouTube videos in one searchable interface.

NYC Insight is a **real-time content aggregation platform** that curates the latest **New York City news articles and YouTube videos** into a single, searchable experience. Built with **React.js** and a **Node.js + Express backend**, the platform integrates multiple external APIs and is **fully deployed in production**.

### Key Highlights
- **API Integration & Backend:** Designed and deployed a custom **Express API** to securely fetch, normalize, and cache data from **NewsAPI** and the **YouTube Data API v3**, protecting API keys and handling server-side errors.
- **Frontend & UX:** Built a responsive, filterable content feed using **React Hooks**, reusable components, and dynamic **search and category filtering** for articles and videos.
- **Architecture & Deployment:** Implemented a **production-ready architecture** with the frontend deployed on **Vercel** and the backend hosted on **Render**, enabling scalable, real-world API consumption.
